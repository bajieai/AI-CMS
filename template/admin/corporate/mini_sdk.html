    SDK模块
    • http — 网络请求(封装wx.request，自动Token刷新)
    • auth — 用户认证(微信登录/Token管理)
    • content — 内容展示(列表/详情/搜索/分类)
    • interaction — 用户交互(收藏/点赞/评论)
    • cache — 缓存管理(TTL过期)
    • share — 分享配置
    • analytics — 数据分析上报
    • utils — 工具函数(日期/截断/价格)
    快速开始
    const AICMS = require('./sdk/ai-cms-sdk.js');
AICMS.init({ baseUrl: 'https://cms.example.com', appId: 'xxx' });
AICMS.content.getList({ model: 'product', page: 1 }).then(res => { console.log(res); });

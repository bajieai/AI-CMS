    上传模板包

    仅支持 .tpkg 格式的模板包文件，最大 50MB
    上传须知
    • 模板包必须包含 manifest.json 描述文件
    • 模板包需通过安全校验后进入人工审核
    • 审核通过后将在模板商店上架
    • 同一 slug 的模板重复上传将作为版本更新处理
    取消

    模板包规范

    manifest.json 必填字段：
    {
  "name": "模板名称",
  "slug": "template-slug",
  "version": "1.0.0",
  "author": "作者",
  "description": "模板描述",
  "category": "企业",
  "screenshots": [],
  "requirements": {
    "php": ">=8.0",
    "cms": ">=2.9.0"
  }
}
    目录结构：
    template-slug/
├── manifest.json
├── pc/
│   ├── index.html
│   └── list.html
├── mobile/
│   └── index.html
├── assets/
│   ├── css/
│   └── js/
└── theme.json

    命令行打包

    也可以使用命令行工具打包模板：

    php think template:package your-theme

    生成的 .tpkg 文件可直接上传。
